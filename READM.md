# 🚀 Task Manager

> Organize. Foque. Conquiste.

Um gerenciador de tarefas moderno, minimalista e responsivo — construído com HTML, CSS e JavaScript puro.

---

## ✨ Features

- **Adicionar, concluir e deletar tarefas** com animações suaves
- **Edição inline** — duplo clique no texto para editar na hora
- **Categorias** — Trabalho 💼, Pessoal 🏠 e Estudo 📚
- **Filtros** por status (Todas / Pendentes / Concluídas) e por categoria
- **Barra de progresso** geral em tempo real
- **Gráfico semanal** de produtividade
- **Relógio e data** ao vivo no header
- **Frases motivacionais** dinâmicas por período do dia
- **Persistência local** via localStorage — suas tarefas ficam salvas
- **Design responsivo** — funciona bem no mobile e desktop

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (variáveis, animações, grid, flexbox)
- JavaScript (ES6+, DOM puro, localStorage)
